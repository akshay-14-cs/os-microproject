How Hit Ratio is Calculated

(Hits ÷ Total Pages) × 100

The Hit Ratio represents the percentage of time the Operating System found the requested page already sitting in the memory frames, avoiding a costly Page Fault.

OS Core Concepts

What is a Page?

A page is a small fixed-size part of a program that is loaded from the hard disk into RAM when needed.

Page म्हणजे प्रोग्रामचा लहान fixed-size भाग जो गरज पडल्यावर Hard Disk वरून RAM मध्ये load केला जातो.

What is a Frame?

A frame is a fixed-size space in RAM where a page is stored.

Frame म्हणजे RAM मधील fixed-size जागा ज्यामध्ये page store केला जातो.

Page Fault

A page fault occurs when the CPU needs a page but it is not present in RAM, so the OS loads it from the hard disk.

Page Fault तेव्हा होतो जेव्हा CPU ला page लागतो पण तो RAM मध्ये नसतो, त्यामुळे Operating System तो Hard Disk वरून RAM मध्ये आणतो.

Page Replacement

Page replacement happens when RAM is full and a new page needs to be loaded. The OS removes one page and loads the new one.

Page Replacement तेव्हा होते जेव्हा RAM full असते आणि नवीन page load करायचा असतो. तेव्हा Operating System एक जुना page काढून नवीन page RAM मध्ये load करते.
